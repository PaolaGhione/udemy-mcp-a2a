# MCP and A2A for Smarter AI Agents

Learning repository for Model Context Protocol (MCP) and Agent-to-Agent (A2A) communication patterns.

## Structure

- `mcp/` — MCP server and client examples
- `a2a/` — Agent-to-Agent protocol examples
- `agent-skills/` — Agent skill implementations
- `labs/` — Exercises and experiments
- `workshop/` — End-to-end projects
